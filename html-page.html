Como Funciona a Estruturação do HTML:

O HTML significa Hyper Text Markup Language, ou seja é a linguagem de marcação padrão para criar páginas da Web. HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.

Logo abaixo vou listar exemplos de tags html de como elas funcionam e onde podemos utiliza-lás em nosso dia a dia na programação, desde de tags mais básicas até as mais avançadas.

Exemplo 1 (html):

A tag <html> é a raiz de um documento HTML e indica ao navegador que o conteúdo dentro dela é um documento HTML.

Exemplo de estrutura básica do HTML:

Para copiar, clique no ícone ao lado

1<!DOCTYPE html> 2<html lang="pt-br"> 3<head> 4 <meta charset="UTF-8"> 5 <title>Document</title> 6</head> 7<body> 8 <!-- Conteúdo do Body --> 9</body> 10</html>

No card acima temos um exemplo da tag html, nas linhas 2 e 10 podemos ver a abertura e fechamento da tag, que delimita o documento. O atributo lang define uma linguagem a página, neste caso em PT-BR. Sendo assim, todas as demais tags da página devem estar neste espaço.

Exemplo 2 (head):

A tag <head> define o cabeçalho do documento, o conteúdo nesse espaço não é visível no browser, mas contém instruções sobre seu conteúdo e comportamento. Dentro dessa tag por exemplo, podem ser inseridas folhas de estilo, scripts, título e metadados. Os navegadores que oferecem suporte a essa tag são: Chrome - Microsoft Edge - Firefox - Safari - Opera.

No primeiro card logo acima, temos um exemplo de uso desta tag, nas linhas 3 e 6 é onde começa a abertura da tag e o fechamento da mesma, o símbolo </> indica o fechamento da tag.

Exemplo 3 (meta):

A tag <meta> pode ser usada acompanhada de alguns atributos, sendo eles o charset ou (character) que é utilizado para definir qual será o conjunto de caracteres para renderizar o texto da página, neste caso foi o UTF-8 que contém todos os caracteres dos padrões Unicode e ASCII, sendo portanto, o mais utilizado em páginas web.

Porém com outros atributos, pode ser utilizada para outros fins, como na SEO (Search Engine Optimization), ou seja é um conjunto de técnicas e estratégias usadas para melhorar a visibilidade de um site nos resultados de busca do Google e outros mecanismos de busca.

Segue abaixo um exemplo usando atributos com palavras chaves para buscas:

Para copiar, clique no ícone ao lado

1<!DOCTYPE html> 2<html lang="pt-br"> 3<head> 4<meta charset="UTF-8"> 5<meta name="author" content="Seu Nome"> 6<meta name="description" content="Exemplo Tag Meta"> 7<meta name="viewport" content="exampleHtml, ads, tagMeta"> 8<meta name="viewport" content="width=device-width, initial-scale=1.0"> 9 <title>Document</title> 10 </head> 11 <body> 12 <!-- Conteúdo --> 13 </body> 14 </html>

Exemplo 4 (link):

A tag <link> especifica as relações entre o documento atual e um recurso externo. Possíveis usos para esta tag incluem a definição de uma estrutura relacional para navegação ou incluir estilos de links externos. Esta tag é mais usada para vincular as folhas de estilo.

Esta tag inclui os atributos como o rel que especifica a relação entre o documento HTML atual e o recurso vinculado. E o href especifica o caminho ou URL do recurso vinculado, ele pode ser um caminho absoluto (URL completo) ou um caminho relativo (relativo à localização atual do documento HTML).

Segue abaixo um exemplo:

Para copiar, clique no ícone ao lado

1<link rel="stylesheet" href="./style/index.css">

No card acima, o link esta sendo utilizado pra vincular a uma folha de estilo.

Exemplo 5 (body):

A tag <body> é um dos elementos mais fundamentais em um documento HTML. Ela representa o corpo do documento e contém todo o conteúdo visível para o usuário no navegador. É o principal contêiner para o conteúdo da página e é fundamental para o layout, interatividade e experiência do usuário no site. Sem essa tag, o navegador não teria um lugar claro para renderizar o conteúdo que deve ser exibido ao usuário.

Segue abaixo um exemplo:

Para copiar, clique no ícone ao lado

1<body> 2<h2>Meu Texto dentro do Body</h2> 3<h3>Lorem, ipsum dolor sit amet consectetur adipisicing elit.</h3> 4</body>

Exemplo 6 (header):

A tag <header> é usada para definir a parte superior ou o cabeçalho de uma página ou seção. Normalmente, o conteúdo do header inclui elementos como logo do site, menus de navegação, títulos, ou qualquer outro tipo de informação introdutória.

Pode ser usada mais de uma vez em uma página, ou seja, você pode ter um header global para o site e outro para seções internas, e também não se limita a conter elementos de navegação.

Aqui esta um exemplo de como ele funciona:

Para copiar, clique no ícone ao lado

1 <header> 2 <h1>Meu Site</h1> 3 <nav> 4 <ul> 5 <li><a href="#home">Home</a></li> 6 <li><a href="#sobre">Sobre</a></li> 7 <li><a href="#contato">Contato</a></li> 8 </ul> 9 </nav> 10 </header>

Exemplo 7 (div):

As divs são normalmente utilizados para representar containers para outros elementos, agrupando-os visualmente dentro de um bloco que pode conter dimensões e posição definidas. Por padrão, uma div não possui aparência características visuais definidas, isso precisa ser feito via CSS ao atribuir bordas, cores etc.

Sua principal característica, no entanto, é que essa tag representa um elemento do tipo bloco, ou seja, que quando adicionado na página, automaticamente gera uma nova linha no layout (semelhante a um parágrafo), ao invés de ser alocado lateralmente nos demais componentes.

Aqui esta um exemplo de como funciona:

Para copiar, clique no ícone ao lado

Exemplo 8 (span):

A tag span é um elemento do tipo inline, ou seja, quando adicionado na página, ele é inserido lateralmente após os demais componentes, diferente das divs que são elementos do tipo bloco.

Elementos span, por padrão, também não possuem nenhuma característica visual definida, isso precisa ser feito via CSS para destacar ou aplicar uma formatação especial para um certo trecho do texto.

No card abaixo, o trecho de código possui uso do span em dois casos. No primeiro, a tag não conta com nenhum atributo adicional, porém no segundo, adicionamos a ela uma aparência diferenciada via CSS.

Aqui esta um exemplo de como funciona:

Para copiar, clique no ícone ao lado

1<p>Este é um texto com um <span> 2trecho em destaque</span></p> 3<p> 4Este é um texto com um <span 5style="background: #9c89e0; border-radius:5px; padding:5px"> 6trecho em destaque</span>.</p>

Exemplo 9 (H1 ao H6):

As tags h1 até h6 são usadas para criar títulos e subtítulos em uma página web. Cada uma dessas tags tem um nível de importância, indo do mais importante ao menos importante. O título mais destacado é h1, e o menos destacado é h6.

Essas tags ajudam a organizar o conteúdo de uma página e melhorar sua estrutura tanto para os leitores quanto para os motores de busca (como o Google), que usam essa hierarquia para entender a relevância do conteúdo.

Aqui esta um exemplo de como funciona:

Para copiar, clique no ícone ao lado

1<h1>Título de nível 1</h1> 2<h2>Título de nível 2</h2> 3<h3>Título de nível 3</h3> 4<h4>Título de nível 4</h4> 5<h5>Título de nível 5</h5> 6<h6>Título de nível 6</h6>

Exemplo 10 (a):

Esta tag A é utilizada para criar links em uma página web, permitindo a navegação entre diferentes páginas ou partes de um documento, ou até mesmo o redirecionamento para outros sites.

A tag a é uma das mais importantes em HTML, pois possibilita a interconexão entre conteúdos na web. Os principais atributos da tag a incluem o "href" que define o destino do link e pode ser um endereço se site completo (url), ou um caminho para um arquivo da mesma página.

Aqui esta um exemplo de como funciona:

Para copiar, clique no ícone ao lado

Exemplo 14 (img):

Esta tag é muito utilizada para inserir imagens em uma página web. Ela é uma tag vazia, ou seja, não possui uma tag de fechamento, e é composta por atributos que ajudam a especificar a imagem, como o caminho da imagem, texto alternativo, tamanho, entre outros. Os principais atributos da tag img incluem:

  • src - que especifica o caminho (url) da imagem;

  • alt - fornece um texto alternativo que descreve a imagem. Esse texto é exibido se a imagem não puder ser carregada e é importante para acessibilidade;

  • widht e height - que definem a largura e a altura da imagem (em pixels, ou como uma porcentagem do tamanho original).

    • Aqui esta um exemplo de como funciona:

    Para copiar, clique no ícone ao lado

    1<img src="gato.jpg" 2     alt="Foto de um gato dormindo" 3     width="400" 4     height="300" 5     title="Gato Dormindo">

    Exemplo 15 (audio):

    Esta tag é usada para incorporar arquivos de áudio em uma página da web. Ela permite ao usuário ouvir um som ou música diretamente no navegador, com controles como play, pause e volume. A estrutura básica da tag audio inclui o atributo controls, que exibe o painel de controle padrão do player. Outros atributos incluem:

  • src - especifica o arquivo de áudio a ser reproduzido. Alternativamente, pode-se usar a tag source dentro da tag audio;

  • autoplay - faz o áudio iniciar automaticamente quando a página é carregada;

  • loop - permite que o áudio seja reproduzido em loop, ou seja, ele recomeça automaticamente após terminar;

  • muted - inicia o áudio no modo mudo.

    • Aqui esta um exemplo de como funciona:

    Para copiar, clique no ícone ao lado

    1<audio controls> 2   <source src="musica.mp3" type="audio/mpeg"> 3   Seu navegador não suporta a tag de áudio. 4</audio>

    Exemplo 16 (video):

    A tag video é usada para incorporar vídeos diretamente em páginas da web, permitindo que os visitantes visualizem o conteúdo sem a necessidade de plug-ins externos. Ela oferece várias opções de controle e atributos para personalizar a experiência de visualização. Os principais atributos incluem:

  • src - define o caminho do arquivo de vídeo;

  • controls - exibe controles de reprodução, como play/pause, volume e progresso do vídeo;

  • autoplay - reproduz automaticamente o vídeo assim que ele é carregado. Importante observar que alguns navegadores podem bloquear essa função;

  • muted - inicia o áudio no modo mudo.

  • loop - faz com que o vídeo reinicie automaticamente ao chegar ao fim;

  • poster - define uma imagem que será exibida enquanto o vídeo não começa a reproduzir.

    Exemplo 17 (form):

    É usada para criar um formulário onde os usuários podem inserir dados. Os formulários são fundamentais em páginas web interativas, permitindo a coleta de informações dos usuários, como nome, email, senha e muito mais. Os atributos incluem:

  • action - define o URL para onde os dados do formulário serão enviados após o envio. Se não for especificado, os dados serão enviados para a mesma URL da página;

  • method - especifica o método HTTP (GET - POST) que será utilizado ao enviar os dados;

  • enctype - ndica como os dados do formulário devem ser codificados quando enviados. É especialmente importante ao enviar arquivos;

  • target - especifica onde a resposta será exibida após o envio do formulário. Pode ter valores como _blank (nova aba), _self (mesma aba) ou _parent (pai do frame).

    • Aqui esta um exemplo de como funciona:

    Para copiar, clique no ícone ao lado

    1<form action="/submit" method="POST"> 2  <label for="name">Nome:</label> 3  <input type="text" id="name" name="name" required> 4  <label for="email">Email:</label> 5  <input type="email" id="email" name="email" required> 6  <input type="submit" value="Enviar"> 7</form>

    Exemplo 20 (textarea):

    Esta tag é usada para criar uma área de texto onde os usuários podem inserir um texto de múltiplas linhas. É especialmente útil para formulários que requerem entradas mais longas, como comentários, descrições ou feedback. Ao contrário de um "input" do tipo "text", que é limitado a uma única linha de entrada, a tag textarea permite que o usuário digite quantas linhas quiser. Os atributos incluem:

  • rows - especifica o número de linhas visíveis na área de texto;

  • cols - especifica a largura da área de texto em termos de número de caracteres visíveis por linha;

  • placeholder - fornece uma dica ou exemplo de texto que desaparece quando o usuário começa a digitar;

  • required - indica que o campo é obrigatório e deve ser preenchido antes do envio do formulário;

  • name - o nome do campo, que será enviado ao servidor quando o formulário for enviado.

    • Aqui esta um exemplo de como funciona:

    Para copiar, clique no ícone ao lado

    1<label for="mensagem">Mensagem:</label> 2<textarea id="mensagem" name="mensagem" 3<rows="4" cols="50" placeholder="Digite sua mensagem aqui..." required> 5</textarea>

    Exemplo 21 (button):

    A tag button em HTML é usada para criar botões que podem ser clicados pelo usuário. Os botões podem ser utilizados para executar ações, como enviar um formulário, disparar uma função em JavaScript ou interagir com outros elementos na página. Os atributos incluem:

  • type - especifica o tipo de botão. Os tipos mais comuns são:
    • (reset) - redefine os valores dos campos do formulário para seus valores iniciais;

    • (button) - botão genérico que não tem um comportamento padrão ou seja não envia um formulário;

    • (submit) - envia o formulário associado quando clicado, este é o tipo padrão se nenhum tipo for especificado;

  • name - define um nome para o botão, que pode ser útil ao enviar dados de formulários;

  • value - define um valor associado ao botão, que é enviado com o formulário quando ele é submetido;

  • disabled - se presente, o botão será desativado e não poderá ser clicado;

  • onclick - um evento JavaScript que é acionado quando o botão é clicado, permitindo a execução de uma função.

    • Aqui esta um exemplo de como funciona:

    Para copiar, clique no ícone ao lado

    1<form> 2  button type="button" onclick="alert('Botão clicado!')">Clique aqui</button> 3</form>

    Exemplo 22 (table):

    Exemplo 23 (article):

    É uma parte semântica da linguagem HTML5 que permite que os desenvolvedores estruturem o conteúdo de forma mais significativa. O article pode conter textos, imagens, vídeos, e outros elementos que têm significado próprio.

    Esta tag é muito utilizada em blogs, sites de notícias, publicações científicas, fóruns e qualquer lugar onde conteúdo independente é apresentado. Isso ajuda tanto na organização do código quanto na acessibilidade e SEO, facilitando a navegação e compreensão do conteúdo por usuários e motores de busca.

    As características desta tag incluem:

    • Independência: O conteúdo dentro de um article deve fazer sentido por conta própria e deve ser capaz de ser distribuído de forma independente. Por exemplo, um post de blog, uma notícia ou um comentário podem ser considerados artigos.

    • Uso em Agrupamentos: A tag article é frequentemente utilizada dentro de seções mais amplas, como section, aside, ou main, para organizar melhor o conteúdo de uma página.

    • SEO e Acessibilidade: O uso de article melhora a estrutura semântica da página, tornando-a mais compreensível para motores de busca e para tecnologias assistivas, como leitores de tela.

    Aqui esta um exemplo de como funciona:

    Para copiar, clique no ícone ao lado

    Exemplo 24 (section):

    Exemplo 25 (nav):

    A tag nav é uma das tags semânticas do HTML5 que é utilizada para definir uma seção de navegação dentro de um documento. Seu objetivo principal é agrupar links de navegação que permitem ao usuário se mover entre diferentes partes de uma página ou para outras páginas do site. O uso da tag nav ajuda tanto na organização do código quanto na acessibilidade, permitindo que tecnologias assistivas identifiquem facilmente a área de navegação.

    Características da tag nav:

    • Semântica: Ao usar nav, você indica que o conteúdo dentro dela está relacionado à navegação, o que ajuda os mecanismos de busca e leitores de tela a entenderem melhor a estrutura da página.

    • Acessibilidade: O uso dessa tag melhora a experiência de navegação para usuários com deficiências, pois leitores de tela podem saltar diretamente para a seção de navegação.

    • Conteúdo: A tag nav pode conter uma variedade de elementos, como listas de links (ul), links diretos (a), ou outras tags que façam referência a seções do site.

    Aqui esta um exemplo de como funciona:

    Para copiar, clique no ícone ao lado

    No exemplo acima, apresentamos a tag nav, que é utilizada para definir uma seção de navegação em um documento HTML. Essa seção contém uma lista de links que orientam os usuários pelas diferentes páginas ou seções do site. A lista é organizada com a tag ul (lista não ordenada), e cada item da lista é criado com a tag li (item de lista).

    Os itens de navegação são apresentados de forma sublinhada, destacando suas funções e facilitando a identificação das opções disponíveis. Esse tipo de estrutura é fundamental para a usabilidade de um site, pois permite que os visitantes naveguem de maneira intuitiva e eficiente.

    Exemplo 26 (footer):

    A tag footer é utilizada para definir o rodapé de um documento ou de uma seção específica de uma página. Essa tag é essencial para a organização do conteúdo e para melhorar a acessibilidade, pois permite agrupar informações que geralmente são relevantes para o usuário, como:

    • Informações de Contato: Ao usar nav, você indica que o conteúdo dentro dela está relacionado à navegação, o que ajuda os mecanismos de busca e leitores de tela a entenderem melhor a estrutura da página.

    • Créditos e Direitos Autorais: O uso dessa tag melhora a experiência de navegação para usuários com deficiências, pois leitores de tela podem saltar diretamente para a seção de navegação.

    • Links de Navegação: A tag nav pode conter uma variedade de elementos, como listas de links (ul), links diretos (a), ou outras tags que façam referência a seções do site.
