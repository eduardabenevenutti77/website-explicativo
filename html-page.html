Como Funciona a Estruturação do HTML:

O HTML significa Hyper Text Markup Language, ou seja é a linguagem de marcação padrão para criar páginas da Web. HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.

Exemplo 1 (html):

A tag <html> é a raiz de um documento HTML e indica ao navegador que o conteúdo dentro dela é um documento HTML.

Exemplo 2 (head):

A tag <head> define o cabeçalho do documento, o conteúdo nesse espaço não é visível no browser, mas contém instruções sobre seu conteúdo e comportamento. Dentro dessa tag, por exemplo, podem ser inseridas folhas de estilo, scripts, título e metadados. Os navegadores que oferecem suporte a essa tag são:
Chrome - Microsoft Edge - Firefox - Safari - Opera.

Exemplo 3 (meta):

A tag <meta> nesse caso, especifica qual conjunto de caracteres (character set ou charset) será usado para renderizar o texto da página. O UTF-8 contém todos os caracteres dos padrões Unicode e ASCII, sendo, portanto, o mais utilizado em páginas web.
