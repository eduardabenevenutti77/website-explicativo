Como funciona a estilização por CSS:

É a linguagem que utilizamos para modelar a aparência dos elementos de uma página web por meio de regras semânticas, de tal maneira, está regras são compostas por propriedades que ao serem aplicadas em um elemento HTML alteram suas características visuais. A regra para declarar o objeto de alteração de um elemento, pode ser realizado da seguinte maneira:

  • Necessitamos do objeto a ser alterado, suponha que seja um elemento de texto, como: parágrafo "p"
  • Ao delimitar tal objeto, necessitamos informar qual vai ser o estilo que será alterado, neste caso, podemos usar como exemplo a cor de um parágrafo.
              
                <!DOCTYPE html>
                <html lang="pt-br">
                <head>
                  <meta charset="UTF-8">
                  <meta name="viewport" content="width=device-width, initial-scale=1.0">
                  <title>Exemplo de Uso CSS</title>
                  <style>
                    p {
                      color: yellow; 
                    }
                  </style>
                </head>
                <body>
                  <p>Exemplo de estilização de uma tag p</p>
                </body>
                </html>

Vale-se ponderar que as propriedades: color, background, font, display, position, width, height, none, block, inline, flex, grid, absolute, relative, fixed, auto, inherit, initial, transparent, red, blue, green, yellow, entre outras, são palavras de uso restrito do CSS. Então, devemos evitar o uso dessas palavras em outras partes do desenvolvimento para evitar conflitos!

Mas como podemos selecionar um elemento para realizar a sua alteração? Atualmente existe três maneiras para realizar a estilização de um determinado elemento. A seguir iremos demonstrar na prática o uso, por meio de: elemento, classe e identificador.

1º - Elemento HTML

Podemos modelar o visual de um elemento html pela sua própria tag, neste caso suponha-se que há um mini código desenvolvido em HTML e desejamos alterar o estilo da fonte, a cor e o tamanho de uma tag 'p'


              <!DOCTYPE html>
              <html lang="pt-br">
              <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Exemplo de Uso CSS</title>
                <style>
                  p {
                    color: yellow; 
                    font-style: italic;
                    font-size: 15px;
                  }
                </style>
              </head>
              <body>
                <p>Exemplo de estilização de uma tag p</p>
              </body>
              </html>

2º - Class

A segunda maneira de alterar visualmente um elemento é por meio da sua classe, isso consiste em elementos que possuem como delimitadores o seguinte caractere: '.'


                <!DOCTYPE html>
                <html lang="pt-br">
                <head>
                  <meta charset="UTF-8">
                  <meta name="viewport" content="width=device-width, initial-scale=1.0">
                  <title>Exemplo de Uso CSS</title>
                  <style>
                    p {
                      color: yellow; 
                      font-style: italic;
                      font-size: 15px;
                    }
                    .fix-color {
                      color: red;
                    }
                  </style>
                </head>
                <body>
                  <p>Exemplo de estilização de uma tag p</p>
                  <p class="fix-color">Exemplo de estilização usando classes</p>
                </body>
                </html>

Todos os elementos que possuírem no HTML a propriedade class 'fix-color' terão sua cor alterada para vermelho. Isso facilitará a vida do programador quando desejar que apenas alguns elementos de sua aplicação tenham determinada estilização.

3º - ID

Existe também a possibilidade de estilizar apenas um único elemento dentro de uma página, mas como isso ocorre? 


              <!DOCTYPE html>
              <html lang="pt-br">
              <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Exemplo de Uso CSS</title>
                <style>
                  p {
                    color: yellow; 
                    font-style: italic;
                    font-size: 15px;
                  }
                  .fix-color {
                    color: red;
                  }
                  #change-id {
                    font-weight: 800;
                    color: #5e42a6;
                  }
                </style>
              </head>
              <body>
                <p>Exemplo de estilização de uma tag p</p>
                <p class="fix-color">Exemplo de estilização usando classes</p>
                <p id="change-id">Exemplo de estilização usando ID</p>
              </body>
              </html>

Com o uso de delimitadores por identificação fica restrito o uso para somente aquele elemento em específico. Podemos grifar pontos extremamentes importantes do uso de ID na estilização, como: Quando existe conflito em nossa estilização, o elemento que possuir id será priorizado. Outro ponto interessante é desempenho em comparação ao uso de classes;

Como funciona a propriedade de texto no CSS:

1ª - Color: Está propriedade define qual cor o nosso elemento de texto irá possuir. Para utilizar, podemos informar os seguintes valores: O nome da cor (`red`), hexadecimais (`#ff000`) e até valores RGB/RGBA(`rgb(255, 0, 0)`)

2ª - Font-size: O tamanho que o texto terá, podemos usar: pixels (`15px`), em (`15em`), porcentagem (`15%`)

3ª - Font-style: O estilo do texto, podendo ser em negrito, normal, itálico e etc

4ª - Text-aling: Define o alinhamento do texto dentro da página, podemos definir como: esquerdo -> left, direito -> right, centralizado -> center, justificado -> justify

5ª - Text-transform: Controla a capitalização do texto, mas o que é isso? É o estilo em qual as frases estarão. De tal modo, podemos deixar tudo em maiúsculas, minúsculas, cada começo de frase e etc

Como funciona a propriedade de layouts no CSS:

Display: Essa propriedade defina a maneira pela qual o nosso elemento será disposto na tela do usuário, dentro dessa propriedade podemos definir em: blocos -> block, linha -> inline, flexível -> flex e em grade -> grid

Position: A posição de um elemento dentro da página, podemos utilizar o valor padrão -> static, relativo -> relative, absoluto -> absolute e por fim podemos utilizar o valor fixo -> fixed

Os elementos top, left, right, bottom define a posição do elemento filho dentro do seu elemento pai, essa posição é amplamente dependente do position delimitado a ele

Já o elemento z-index delimita a ordem de empilhamento dos elementos. Conforme mais alto for o valor informado a ele, mais perto da tela ele estará!

Como funciona a propriedade do box model no CSS:

1ª Margin: Define o espaçamento ao redor do elemento;

2ª Padding: Define o espaçamento interno do elemento;

3ª Border: Define a borda de um elemento, podendo ter um estilo, cor e espessura;

4ª Width e Height: Definem a largura e a altura de um elemento;

5ª Box-sizing: Define como a largura e a altura de elemento serão calculadas, pode ser utilizado as seguintes opções: border-box -> inclui o padding e borda, e o content-box -> esse não possui nada junto;

Como funciona a propriedade de background no CSS:

Background-color: A cor de fundo do elemento;

Background-image: Define uma imagem para o fundo;

Background-size: Define como a imagem irá ocupar o elemento;

Background-position: Delimita a posição da imagem de fundo;

Background-repeat: Define se a imagem do fundo será repetida, e a posição: repeat-x -> linha e repeat-y -> coluna

Como funciona a propriedade Flexbox no CSS

Display: flex | Delimita o container como flexbox, o que permite o alinhamento e distribuição flexíveis dos elementos filhos

Justify-content: Delimita a maneira pela qual os elementos filhos serão distribuídos no eixo principal

Align-items: O alinhamento dos elementos filhos ao longo do eixo transversal

Flex-wrap: Define se os itens devem quebrar para uma nova linha quando não houver espaço suficiente -> wrap ou permanecerem em uma linha -> nowrap

Como funciona a propriedade Grid no CSS

Display: grid Define um contêiner como um Grid, permitindo criar layouts baseados em colunas e linhas

Grid-template-columns Define o número de colunas e a largura de cada uma

Grid-template-rows Define o número de linhas e a altura de cada uma

Gap Define o espaçamento entre colunas e/ou linhas
