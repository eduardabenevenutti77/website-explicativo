Como funciona a estilização por CSS:

É a linguagem que utilizamos para modelar a aparência dos elementos de uma página web por meio de regras semânticas, de tal maneira, está regras são compostas por propriedades que ao serem aplicadas em um elemento HTML alteram suas características visuais. A regra para declarar o objeto de alteração de um elemento, pode ser realizado da seguinte maneira:

  • Necessitamos do objeto a ser alterado, suponha que seja um elemento de texto, como: parágrafo "p"
  • Ao delimitar tal objeto, necessitamos informar qual vai ser o estilo que será alterado, neste caso, podemos usar como exemplo a cor de um parágrafo.
              
                <!DOCTYPE html>
                <html lang="pt-br">
                <head>
                  <meta charset="UTF-8">
                  <meta name="viewport" content="width=device-width, initial-scale=1.0">
                  <title>Exemplo de Uso CSS</title>
                  <style>
                    p {
                      color: yellow; 
                    }
                  </style>
                </head>
                <body>
                  <p>Exemplo de estilização de uma tag p</p>
                </body>
                </html>

Vale-se ponderar que as propriedades: color, background, font, display, position, width, height, none, block, inline, flex, grid, absolute, relative, fixed, auto, inherit, initial, transparent, red, blue, green, yellow, entre outras, são palavras de uso restrito do CSS. Então, devemos evitar o uso dessas palavras em outras partes do desenvolvimento para evitar conflitos!

Mas como podemos selecionar um elemento para realizar a sua alteração? Atualmente existe três maneiras para realizar a estilização de um determinado elemento. A seguir iremos demonstrar na prática o uso, por meio de: elemento, classe e identificador.

1º - Elemento HTML

Podemos modelar o visual de um elemento html pela sua própria tag, neste caso suponha-se que há um mini código desenvolvimento em HTML e desejamos alterar o estilo da fonte, a cor e o tamanho de uma tag 'p'


              <!DOCTYPE html>
              <html lang="pt-br">
              <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Exemplo de Uso CSS</title>
                <style>
                  p {
                    color: yellow; 
                    font-style: italic;
                    font-size: 15px;
                  }
                </style>
              </head>
              <body>
                <p>Exemplo de estilização de uma tag p</p>
              </body>
              </html>

2º - Class

A segunda maneira de alterar visualmente um elemento é por meio da sua classe, isso consiste em elementos que possuem como delimitadores o seguinte caractere: '.'


                <!DOCTYPE html>
                <html lang="pt-br">
                <head>
                  <meta charset="UTF-8">
                  <meta name="viewport" content="width=device-width, initial-scale=1.0">
                  <title>Exemplo de Uso CSS</title>
                  <style>
                    p {
                      color: yellow; 
                      font-style: italic;
                      font-size: 15px;
                    }
                    .fix-color {
                      color: red;
                    }
                  </style>
                </head>
                <body>
                  <p>Exemplo de estilização de uma tag p</p>
                  <p class="fix-color">Exemplo de estilização usando classes</p>
                </body>
                </html>

Todos os elementos que possuírem no HTML a propriedade class 'fix-color' terão sua cor alterada para vermelho. Isso facilitará a vida do programador quando desejar que apenas alguns elementos de sua aplicação tenham determinada estilização.

3º - ID

Existe também a possibilidade de estilizar apenas um único elemento dentro de uma página, mas como isso ocorre? 


              <!DOCTYPE html>
              <html lang="pt-br">
              <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Exemplo de Uso CSS</title>
                <style>
                  p {
                    color: yellow; 
                    font-style: italic;
                    font-size: 15px;
                  }
                  .fix-color {
                    color: red;
                  }
                  #change-id {
                    font-weight: 800;
                    color: #5e42a6;
                  }
                </style>
              </head>
              <body>
                <p>Exemplo de estilização de uma tag p</p>
                <p class="fix-color">Exemplo de estilização usando classes</p>
                <p id="change-id">Exemplo de estilização usando ID</p>
              </body>
              </html>

Com o uso de delimitadores por identificação fica restrito o uso para somente aquele elemento em específico. Podemos grifar pontos extremamentes importantes do uso de ID na estilização, como: Quando existe conflito em nossa estilização, o elemento que possuir id será priorizado. Outro ponto interessante é desempenho em comparação ao uso de classes;
