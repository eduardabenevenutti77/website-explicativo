Introdução ao React

React é uma biblioteca JavaScript de código aberto usada para construir interfaces de usuário, especialmente para aplicações web de página única. Criado pelo Facebook, ele facilita o desenvolvimento de UIs reutilizáveis e escaláveis.

1. Estrutura de Componentes

Os componentes são as unidades fundamentais do React. Eles podem ser reutilizados para criar interfaces de usuário dinâmicas. Um componente pode ser uma função ou classe que retorna elementos React.

Exemplo com componente funcional:


function Welcome(props) {
  return <h1>Olá, {props.name}!</h1>;
}

const App = () => (
  <div>
    <Welcome name="Maria" />
    <Welcome name="João" />
  </div>
);

Exemplo com componente de classe:


import React, { Component } from 'react';

class Saudacao extends Component {
  render() {
    return <h1>Olá, {this.props.name}!</h1>;
  }
}

export default Saudacao;

2. Hooks

Hooks permitem adicionar funcionalidades como estado e efeitos colaterais em componentes funcionais. Os mais comuns são useState e useEffect.

Exemplo de uso do useState:


import React, { useState } from 'react';

function Contador() {
  const [contador, setContador] = useState(0);

  return (
    <div>
      <p>Você clicou {contador} vezes</p>
      <button onClick={() => setContador(contador + 1)}>Clique aqui</button>
    </div>
  );
}

Exemplo de uso do useEffect:


import React, { useState, useEffect } from 'react';

function Relogio() {
  const [hora, setHora] = useState(new Date());

  useEffect(() => {
    const timer = setInterval(() => setHora(new Date()), 1000);
    return () => clearInterval(timer);
  }, []);

  return <p>Hora atual: {hora.toLocaleTimeString()}</p>;
}

Exemplo de uso do useReducer:


import React, { useReducer } from 'react';

function reducer(state, action) {
  switch (action.type) {
    case 'increment': return { count: state.count + 1 };
    case 'decrement': return { count: state.count - 1 };
    default: return state;
  }
}

function Contador() {
  const [state, dispatch] = useReducer(reducer, { count: 0 });

  return (
    <div>
      <p>Contagem: {state.count}</p>
      <button onClick={() => dispatch({ type: 'increment' })}>+</button>
      <button onClick={() => dispatch({ type: 'decrement' })}>-</button>
    </div>
  );
}

3. JSX

JSX é uma extensão de sintaxe que permite escrever código similar ao HTML dentro de arquivos JavaScript. Ele facilita a criação de elementos React.

Exemplo de uso básico de JSX:


const elemento = <h1>Bem-vindo ao React!</h1>;
ReactDOM.render(elemento, document.getElementById('root'));

4. React Router

React Router é uma biblioteca que permite a navegação entre páginas em uma aplicação React. Ela cria rotas para diferentes componentes. 


import { BrowserRouter as Router, Route, Switch } from 'react-router-dom';

function App() {
  return (
    <Router>
      <Switch>
        <Route path="/" exact component={Home} />
        <Route path="/sobre" component={Sobre} />
      </Switch>
    </Router>
  );
}

5. Gerenciamento de Estado

O gerenciamento de estado é essencial para manter a consistência de dados em uma aplicação React. Ferramentas como Redux ou Context API são utilizadas para estados globais.

O Redux é uma biblioteca de gerenciamento de estado previsível para aplicações JavaScript. Ele centraliza o estado de uma aplicação em um único local, o "store", e fornece uma maneira previsível de gerenciar suas alterações. Em vez de atualizar diretamente o estado, as mudanças são feitas por meio de "actions" e "reducers", facilitando o rastreamento e a depuração da aplicação.

Abaixo, um exemplo de como o Redux pode ser utilizado para gerenciar um contador. O estado é mantido no "store", e as ações de incremento e decremento controlam a modificação desse estado. O componente `App` acessa o estado global por meio do `useSelector` e dispara as ações utilizando o `useDispatch`.

Exemplo com Redux:


import { createStore } from 'redux';
import { Provider, useSelector, useDispatch } from 'react-redux';

// Reducer
function contador(state = 0, action) {
  switch (action.type) {
    case 'increment': return state + 1;
    case 'decrement': return state - 1;
    default: return state;
  }
}

const store = createStore(contador);

function App() {
  const contador = useSelector(state => state);
  const dispatch = useDispatch();

  return (
    <div>
      <p>Contador: {contador}</p>
      <button onClick={() => dispatch({ type: 'increment' })}>+</button>
      <button onClick={() => dispatch({ type: 'decrement' })}>-</button>
    </div>
  );
}

// Envolvendo a Aplicação:
export default function Root() {
  return (
    <Provider store={store}>
      <App />
    </Provider>
  );
}

Contextos: Compartilhando Informações Globais

Um contexto provê informações através de um provedor (provider), tornando-as acessíveis a todos os elementos filhos por meio de um consumidor (consumer).

Exemplo com Context API:


import React, { createContext, useContext } from 'react';

// Criando um contexto
const ThemeContext = createContext();

// Componente provedor do contexto
const ThemeProvider = ({ children }) => {
  const theme = 'dark';

  return (
    <ThemeContext.Provider value={theme}>
      {children}
    </ThemeContext.Provider>
  );
};

// Componente consumidor do contexto
const ThemedComponent = () => {
  const theme = useContext(ThemeContext);

  return <p style={{ color: theme === 'dark' ? 'white' : 'black' }}>Texto com tema dinâmico</p>
};

// Utilizando o contexto
const App = () => {
  return (
    <ThemeProvider>
      <ThemedComponent />
    </ThemeProvider>
  );
};
