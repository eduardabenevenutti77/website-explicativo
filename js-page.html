Entendendo o JavaScript

JavaScript é uma linguagem de programação leve, interpretada e orientada a objetos. Ela permite adicionar interatividade a páginas web e é amplamente usada no desenvolvimento de front-end e back-end, através do Node.js.

1. Sintaxe Básica

A sintaxe do JavaScript é simples, e podemos começar com um exemplo básico que exibe uma mensagem no console do navegador:


  // Exibe uma mensagem no console
  console.log('Olá, JavaScript!');

2. Variáveis e Tipos de Dados

No JavaScript, você pode criar variáveis usando var, let ou const. Veja o exemplo abaixo: 


            let nome = 'João';
            const idade = 25;
            var cidade = 'São Paulo';

As variáveis podem armazenar diferentes tipos de dados, como números, strings, arrays, e objetos.

2.1 Tipos

Os tipos em JavaScript podem ser divididos em duas categorias:

  • Tipos primitivos
  • Tipos de objeto

Os tipos primitivos incluem números, strings e valores booleanos.

Os valores especiais null e undefined são valores primitivos, mas não são números, nem strings e nem booleanos. Qualquer valor em JavaScript que não seja número, string, booleano, null ou undefined é um objeto.

O operador typeof é usado para verificar o tipo de uma variável ou valor.


  typeof "foo"; // 'string'  
  typeof 123; // 'number'

2.2 Números

Ao contrário de muitas linguagens, JavaScript não faz distinção entre valores inteiros e valores em ponto flutuante. Todos os números em JavaScript são representados como valores em ponto flutuante.

Exemplos de literais inteiros:


  0
  1
  100

Exemplos de literais em ponto flutuante:


  3.14
  100.09
  0.33333

2.3 Strings

Para incluir uma string literal em um programa JavaScript, basta colocar os caracteres da string entre aspas simples ou duplas.


  "testing"
  'other testing'

Para concatenar strings, utilizamos o operador +.


  var msg = "Hello " + "world";// "Hello world"
  msg = "Welcome to my blog, " + nome;

A propriedade length determina o tamanho da string.


  "palavra".length // 7

2.4 Booleanos

Os valores booleanos são representados por true e false.

2.5 Null e Undefined

A palavra chave null indica a ausência de um valor. Já o valor undefined representa uma ausência mais profunda, sendo o valor de variáveis que não foram inicializadas.

2.6 Conversão

JavaScript é muito flexível quanto aos tipos de valores que exige. As variáveis em JavaScript são não tipadas, permitindo atribuir diferentes tipos de valores a uma mesma variável.

JavaScript converte valores de um tipo para outro automaticamente, como, por exemplo, números para strings ou valores para booleanos, conforme o contexto.

2.7 Condicional - if / else


  if (n == 1) {
      // executa este bloco if 
  } else if (n == 2) {
     // executa este bloco else if 
  } else {
     // executa este bloco else 
  }

2.8 Operador Condicional Ternário

É possível obter resultados semelhantes usando o operador condicional ternário condition ? expr1 : expr2. 


  resultado = (a > b) ? "a é maior que b" : "b é maior que a";

3. Funções

Funções no JavaScript são blocos de código reutilizáveis que realizam uma ação. Elas podem ser declaradas e invocadas conforme necessário:


        function saudacao(nome) {
          return 'Olá, ' + nome + '!';
        }
        console.log(saudacao('Maria'));

3.1 Declaração de Função

As funções podem ser declaradas com a palavra-chave function:


        function exemplo() {
          console.log('Esta é uma função declarada.');
        }

3.2 Funções Anônimas

Funções anônimas são funções sem nome, frequentemente atribuídas a variáveis:


        const saudacao = function(nome) {
          return 'Olá, ' + nome + '!';
        };
        console.log(saudacao('Lucas'));

3.3 Arrow Functions

Introduzidas no ES6, as arrow functions permitem uma sintaxe mais curta e não têm seu próprio this:


        const soma = (a, b) => a + b;
        console.log(soma(5, 3));

3.4 Parâmetros e Argumentos Padrão

Funções podem ter valores padrão para parâmetros, caso nenhum argumento seja passado:


        function saudacao(nome = 'Visitante') {
          return 'Olá, ' + nome + '!';
        }
        console.log(saudacao()); // "Olá, Visitante!"

3.5 Escopo e Contexto (this)

O escopo define a visibilidade das variáveis, e o this refere-se ao contexto em que a função foi chamada. Em arrow functions, o this é herdado do escopo onde a função foi criada.


        const pessoa = {
          nome: 'Ana',
          saudacao: function() {
            console.log('Olá, ' + this.nome);
          }
        };
        pessoa.saudacao(); // "Olá, Ana"

3.6 Funções de Ordem Superior

Funções de ordem superior recebem funções como argumentos ou retornam funções. Exemplo: 


        function executar(func) {
          func();
        }
        executar(() => console.log('Executando uma função passada como argumento.'));

3.7 Métodos Úteis para Funções

Métodos como call, apply e bind permitem manipular o contexto e invocar funções de maneira flexível:


        const pessoa = { nome: 'Carlos' };
        function apresentar() {
          console.log('Olá, eu sou ' + this.nome);
        }
        apresentar.call(pessoa); // "Olá, eu sou Carlos"

4. Manipulação de DOM

Uma das principais funcionalidades do JavaScript no front-end é a manipulação do DOM (Document Object Model). O DOM é uma interface de programação que permite que programas e scripts interajam com o conteúdo, a estrutura e os estilos de uma página web. Com a manipulação do DOM, é possível acessar e modificar elementos da página HTML de forma dinâmica.

O DOM representa a página como uma árvore de nós, onde cada elemento HTML é um objeto que pode ser manipulado. Cada tag HTML (como <div>, <p>, <a>) é transformada em um nó, e você pode interagir com esses nós por meio do JavaScript. A manipulação do DOM permite alterar conteúdos, modificar estilos, criar novos elementos e até responder a eventos do usuário.

Abaixo estão algumas das operações mais comuns realizadas com a manipulação do DOM:

4.1 Acessar elementos no DOM

Para manipular um elemento, primeiro você precisa acessá-lo. Existem várias maneiras de acessar um elemento, como por id, classe, nome da tag ou até mesmo usando seletores CSS. 


            // Acessar um elemento pelo id
            const elementoPorId = document.getElementById('titulo');
    
            // Acessar elementos por classe (retorna uma coleção de elementos)
            const elementosPorClasse = document.getElementsByClassName('minha-classe');
    
            // Acessar elementos por tag (retorna uma coleção de elementos)
            const elementosPorTag = document.getElementsByTagName('p');
    
            // Acessar um elemento com o seletor CSS (apenas o primeiro encontrado)
            const elementoPorSelector = document.querySelector('.minha-classe');
    
            // Acessar todos os elementos que correspondem ao seletor
            const todosElementosPorSelector = document.querySelectorAll('.minha-classe');

4.2 Modificar conteúdo de elementos

Depois de acessar um elemento, você pode modificar seu conteúdo, seja o texto, o HTML ou os atributos. Aqui estão algumas formas de alterar o conteúdo de um elemento: 


            // Alterar o texto de um elemento com id 'titulo'
            document.getElementById('titulo').innerText = 'Novo Título';
    
            // Alterar o conteúdo HTML de um elemento
            document.getElementById('container').innerHTML = 'Texto HTML alterado!';
    
            // Alterar o texto sem formatação
            document.getElementById('paragrafo').textContent = 'Texto atualizado sem formatação!';

4.3 Alterar atributos de elementos

Você também pode alterar os atributos de um elemento, como o src de uma imagem ou o href de um link, utilizando os métodos setAttribute e getAttribute. 


            // Alterar o atributo 'src' de uma imagem
            document.getElementById('minhaImagem').setAttribute('src', 'novo-caminho-da-imagem.jpg');
    
            // Alterar o atributo 'href' de um link
            document.getElementById('meuLink').setAttribute('href', 'https://www.novosite.com');
    
            // Obter o valor do atributo 'href'
            const linkHref = document.getElementById('meuLink').getAttribute('href');

4.4 Adicionar e remover elementos

Você pode adicionar novos elementos à página ou remover elementos existentes usando métodos como createElement, appendChild, e removeChild. 


            // Criar um novo elemento 
            const novoParagrafo = document.createElement('p');
            novoParagrafo.textContent = 'Este é um novo parágrafo!';
    
            // Adicionar o novo parágrafo ao final do body
            document.body.appendChild(novoParagrafo);
    
            // Remover um elemento
            const elementoARemover = document.getElementById('elementoParaRemover');
            document.body.removeChild(elementoARemover);

4.5 Manipular estilos

A manipulação de estilos também é uma parte importante do DOM. Você pode alterar as propriedades CSS diretamente através da propriedade style. 


            // Alterar a cor de fundo de um elemento
            document.getElementById('meuElemento').style.backgroundColor = 'yellow';
    
            // Alterar o tamanho da fonte de um elemento
            document.getElementById('meuElemento').style.fontSize = '20px';
    
            // Alterar a cor do texto de um elemento
            document.getElementById('meuElemento').style.color = 'red';

4.6 Trabalhando com eventos

Uma das maiores vantagens da manipulação do DOM é a capacidade de interagir com os usuários. Você pode adicionar ouvintes de eventos para detectar ações do usuário, como cliques, movimentos do mouse, teclas pressionadas, etc. 


            // Adicionar um evento de clique a um botão
            document.getElementById('meuBotao').addEventListener('click', function() {
            alert('Botão foi clicado!');
    });
    
          // Adicionar um evento de mouseover a um elemento
          document.getElementById('meuElemento').addEventListener('mouseover', function() {
          this.style.color = 'blue'; // Mudar a cor do texto quando o mouse passar por cima
    });

4.7 Trabalhando com eventos

id="reduce">reduce()

O método reduce() aplica uma função em um acumulador e em cada elemento de um array para reduzi-lo a um único valor.


const numeros = [1, 2, 3, 4];
const soma = numeros.reduce((acumulador, num) => acumulador + num, 0);
console.log(soma); // 10

5. Eventos

JavaScript permite a criação de páginas interativas respondendo a eventos disparados pelo usuário, como cliques, teclas pressionadas ou o movimento do mouse. Um evento é uma ação que ocorre em uma página, e o JavaScript permite que você escute esses eventos e execute funções em resposta.

5.1 Manipulação de Eventos

Abaixo estão alguns exemplos de como manipular eventos em JavaScript: 


  // Exemplo de evento de clique em um botão
  const botao = document.getElementById('meuBotao');
  botao.addEventListener('click', function() {
    alert('Botão clicado!');
  });
  
  // Exemplo de evento de mouse sobre um elemento
  const elemento = document.getElementById('meuElemento');
  elemento.addEventListener('mouseover', function() {
    elemento.style.backgroundColor = 'yellow';
  });
  
  // Exemplo de evento de tecla pressionada
  document.addEventListener('keydown', function(evento) {
    if (evento.key === 'Enter') {
      console.log('Tecla Enter pressionada');
    }
  });

5.2 Eventos Comuns

Alguns dos eventos mais comuns em JavaScript incluem:

  • click: Disparado quando o usuário clica em um elemento.
  • mouseover: Disparado quando o usuário passa o mouse sobre um elemento.
  • mouseout: Disparado quando o usuário move o mouse para fora de um elemento.
  • keydown: Disparado quando o usuário pressiona uma tecla.
  • load: Disparado quando a página termina de carregar.

Esses eventos, combinados com as funções de manipulação de DOM, permitem que você crie interfaces interativas e dinâmicas.

5.3 Async/Await

Você pode usar async/await para tornar o código assíncrono mais legível e fácil de entender.


async function obterDados() {
  try {
    const resposta = await fetch('https://api.exemplo.com/dados');
    const dados = await resposta.json();
    console.log(dados);
  } catch (erro) {
    console.error('Erro:', erro);
  }
}
obterDados();

6. Fetch API

A Fetch API é usada para fazer requisições HTTP assíncronas (por exemplo, obter dados de um servidor). Veja um exemplo básico de como utilizar a Fetch API para obter dados de uma API externa:


            // Faz uma requisição GET a uma API
            fetch('https://api.example.com/dados')
              .then(response => response.json())
              .then(data => console.log(data))
              .catch(error => console.error('Erro:', error));

7. Promessas

As promessas permitem lidar com operações assíncronas. Elas têm três estados: pendente, cumprida e rejeitada. Veja um exemplo:


            function obterDados() {
              return new Promise((resolve, reject) => {
                let sucesso = true;
                if (sucesso) {
                  resolve('Dados recebidos com sucesso!');
                } else {
                  reject('Erro ao obter dados');
                }
              });
            }

            obterDados()
              .then(mensagem => console.log(mensagem))  // Quando a Promise é resolvida
              .catch(erro => console.error(erro));      // Quando a Promise é rejeitada
